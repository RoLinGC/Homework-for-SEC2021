# <label style="color:red">简易计算器
## 功能介绍
* 0-9：数字键
* .：小数点
* +：加法
* -：减法
* *：乘法
* /：除法
* =：计算结果
* 1/x：取倒数
* %：取余数
* del：退格
* CE：清空当前
* C：清空所有
* +/-：取反
* sqrt：开根号
* MC：将当前记忆数字归零
* MR：显示当前计算的数字
* MS：无视目前记忆多少数字，以当前数字取代
* M+：记忆当前数字
* M-：记忆当前数字（负数）
